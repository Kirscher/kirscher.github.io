# kirscher.github.io

Personal academic website of Tristan Kirscher, built with [al-folio](https://github.com/alshedivat/al-folio).

## Local Development

```bash
# Using Docker (recommended)
docker compose up

# Or with Ruby/Jekyll
bundle install
bundle exec jekyll serve
```

## Deployment

Push to `github.com/Kirscher/kirscher.github.io` and GitHub Actions will automatically build and deploy the site.
